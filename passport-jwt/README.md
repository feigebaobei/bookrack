## passport-jwt

使用jsonwebtoken验证用户。

### 配置策略

    let jwtStrategy = require('passport-jwt').Strategy
    passport.use(new jwtStrategy(options, verifyFn))

options是一个包含了如何从req中取出token/验证token的各个设置的对象.
